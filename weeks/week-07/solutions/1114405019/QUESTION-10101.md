# UVA 10101 - Bangla Numbers

## 題目描述

Bangla numbers are used in Bangladesh. They use ',' to separate the numbers in a different way. In Bangladesh, a number is separated in a way that the last portion is 3 digits and the other portions are 2 digits from the right.

## 輸入說明

The input file contains several lines of input. Each line contains a non-negative integer. The integer can have up to 15 digits.

## 輸出說明

For each line of input, print the number in Bangla format.

## 範例輸入

```
123456789
123
1234
12345
123456
1234567
12345678
123456789
```

## 範例輸出

```
12,34,56,789
123
1,234
12,345
123,456
1,234,567
12,345,678
12,34,56,789
```
