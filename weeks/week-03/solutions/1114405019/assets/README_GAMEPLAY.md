# Gameplay Screenshot Instructions

To create the required `assets/gameplay.png` screenshot:

1. Run `python robot_game_MVP.py` from the `1114405019/` directory
2. The pygame window should open showing the robot simulator
3. Use your system's screenshot tool (e.g., Windows Snip & Sketch, Print Screen) to capture the window
4. Save the screenshot as `gameplay.png` in the `assets/` directory
5. The screenshot should show:
   - The grid map
   - Robot position and direction
   - Any scent markers (if present)
   - HUD with status information
   - Control instructions

Make sure the window is fully visible in the screenshot.
