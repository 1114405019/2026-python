# Python Solutions - Student ID: 1114405019

This directory contains solutions for five UVA/ZeroJudge problems as part of the Week 5 assignment.

## Folder Structure

Each question is organized into its own folder containing multiple versions of the solution:

- `q[ID].py`: Optimized standard solution.
- `q[ID]-detail.py`: Solution with line-by-line explanations.
- `q[ID]-easy.py`: Beginner-friendly version using basic logic.
- `q[ID]-Hand-typed.py`: A clean version simulating manual coding style.
- `test_q[ID].py`: Test script for verifying the solution against sample cases.

## Included Problems

1. **q10041**: Vito's Family (Finding the median for minimum distance).
2. **q10050**: Hartals (Simulating party strikes).
3. **q10055**: Monotonicity (Range composition of increasing/decreasing functions).
4. **q10056**: What is the Probability? (Infinite geometric series).
5. **q10057**: A Mid-summer Night's Dream (Finding optimal medians).

## How to Run Tests

Navigate to a question folder and run the test script:
```bash
python test_q10041.py
```
